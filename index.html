About the App

Ledgr-Journal is a personal finance app that helps users record income and expenses, review spending statistics, and optionally sync data with Google Sheets.

Google API Usage

When users connect Google, the app requests access only for these purposes:

  • Read a list of user spreadsheets so a target sheet can be selected.
  • Read and write transaction rows in the selected spreadsheet.

The app does not use Google data for advertising and does not sell personal data.
